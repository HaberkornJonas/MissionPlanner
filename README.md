How to compile
==============

1. Install Git (the distribution you prefer)
2. Install Visual Studio and .NET Framework 4.6.1
        http://www.visualstudio.com/downloads/download-visual-studio-vs
3. Clone project
4. Open MissionPlanner.sln with Visual Studio.
5. Build and run.
